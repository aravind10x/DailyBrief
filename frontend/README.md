# Frontend

This frontend is the primary public demo surface for Daily Brief.

## Purpose

It showcases a founder-focused workflow:

- enter or edit founder context
- review a morning brief with rationale
- approve the brief into execution
- move tasks through the day and capture one learning signal

## Run

```bash
cd frontend
corepack enable
yarn install
yarn dev
```

For the full project story and architecture, see [README.md](../README.md).
